What's inside
🔐
Sessions in MySQL

32-byte random tokens, 24h TTL, FK cascade on user delete, hourly purge of expired rows.

🧭
Role-based routing

RequireRole middleware returns 403 on mismatch. Login redirect goes to the role's home page.

🎨
Three layouts

Public top-nav, admin sidebar, seller sidebar — chosen automatically by template path.

📦
Self-contained binary

Templates embedded via //go:embed. No template files to deploy alongside.

Stack
  • Go · standard net/http + html/template
  • MySQL · go-sql-driver/mysql with connection pooling
  • godotenv · .env loaded at startup, real env wins
  • bcrypt · password hashing via golang.org/x/crypto
